# Customer Segmentation & Revenue Analysis

## Project Overview
This project focuses on analyzing e-commerce transactional data to understand customer purchasing behavior and segment customers based on their revenue contribution. The goal is to identify high-value customers and analyze revenue distribution across customer segments.

---

## Objectives
- Clean and prepare raw transactional data for analysis  
- Calculate revenue from quantity and unit price  
- Segment customers into High, Medium, and Low value groups  
- Build an interactive Power BI dashboard for business insights  

---

## Tools & Technologies
- Microsoft Excel (Data Cleaning, Data Preparation)
- Power BI (Dashboard & Visualization)
- DAX (Measures and Calculations)
